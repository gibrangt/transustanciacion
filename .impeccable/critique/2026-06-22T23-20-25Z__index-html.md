## Priority Issues

**[P2] Section 7 kicker duplicates the title**
Kicker: "Videos del debate" / Title: "Videos del Debate" — the same phrase in different case. Every other section has distinct thematic orientation: "Dónde chocan" / "Los tres ejes del conflicto", "Postura católica" / "Los argumentos de Urbina". This one reads as an unfilled placeholder visible against the otherwise consistent system.
*Fix*: Change the kicker to something thematically distinct — "El debate en movimiento" or "Mira y escucha" would provide orientation rather than repeat the title.
*→ `/impeccable polish`*

**[P2] Respuestas de Pacheco group contains inflammatory titles without a curatorial note**
The fix added a note only to Terceros. The Pacheco group shows "INCREÍBLE! Dante Urbina LLORA EN VIVO" and "IDOLATRÍA a la HOSTIA" without framing. These appear before the Terceros note and represent the same brand-voice risk — a skeptical reader sees polemics first.
*Fix*: Add an equivalent note inside the Respuestas de Pacheco `<details>`: "Los títulos reflejan la postura del canal de Edgar Pacheco."
*→ `/impeccable clarify`*

**[P2] 13 of 17 panels still show "Fuente por añadir"**
The primary audience job-to-be-done is finding citable patristic sources. 76% of panels fail this. No design change can resolve it, but it remains the highest-impact friction for the target user.
*Fix*: Content priority — complete source links for Arguments 1, 2, 3 (the most-cited patristic arguments) first.

**[P3] Deep links (#arg-1, #obj-3) functional but never discoverable**
Marco (the apologetics user) wants to share a specific argument. The URL mechanism works but there is no affordance. No share icon, no "enlace a este argumento" in any panel header.
*Fix*: Small `<a href="#arg-N">` permalink icon in each `.detalle__header`, styled like `.fuente-tag`. Pure HTML/CSS, no JavaScript required.
*→ `/impeccable polish`*
